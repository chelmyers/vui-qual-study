Memos

User will use GUI over VUI is the device has a screen and is already in hand. Why is this? Is it a trust issue? User notes they do not what to disrupt what they are currently doing.

Bias from assuming Alexa has same features as Siri.

When using commands that are new, users are delighted when it works.

Alexa actually never turned on for first attempt at task 3. User missed this since they were not looking at Alexa. User assumed Alexa did not understand his phrase then.

When VUI fails, user often think they need to be more specific to get it to work.

User believes news should presented like headlines. This is also how he reads it habitual right now.
